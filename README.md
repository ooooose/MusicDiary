# MusicDiary

1日を音楽と共に振り返ろう

## サービス概要
「Music Diary」は日記を書くごとに文章に適した楽曲を提供するアプリです。<br />
1日の振りかえりを行い、日記を作成するとニッキの内容に応じて適切な楽曲を提案します。<br />

## ユーザー層について
日記を書く習慣をつけたいが、なかなかうまくいかないといった方を対象としています。<br />
